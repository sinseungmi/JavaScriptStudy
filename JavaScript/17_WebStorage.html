# Web Storage API

  • 웹 브라우저에서 제공하는 약간의 저장 공간
  • 사용자의 웹 브라우저에 웹 페이지를 운용하는데 필요한 약간의 데이터를 저장해둘 수 있다
  • key:Value 형태
  • localStorage는 웹 브라우저를 종료하더라도 데이터가 유지된다
  • 세션 스토리지는 웹 브라우저를 종료하면 데이터가 삭제된다

# 로컬 스토리지에 저장되는 게임

  1. 자바스크립트로 동작하는 간단한 게임을 하나 만든다
  2. 로컬 스토리지에 사용자 이름과 진행 상황이 저장된다
  3. 다시 접속하더라도 저장된 데이터를 이용하여 게임이 이어서 진행되 도록 만들어보세요
